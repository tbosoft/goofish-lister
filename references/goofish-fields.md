# Goofish publish fields (notes)

This file captures the *observable* fields on the Goofish web publish form as we discover them.

## URLs
- Home: https://www.goofish.com/
- Search: https://www.goofish.com/search?q=<keyword>
- Publish: https://www.goofish.com/publish
- IM: https://www.goofish.com/im

## Publish form (to confirm)
Typical fields to locate by visible labels / placeholders:
- Title
- Description
- Price
- Category
- Condition
- Shipping (if present)
- Location
- Images upload

## Anti-bot / safety
- Prefer real Chrome profile (logged in).
- Avoid headless for publish.
- Never click final “Publish” automatically.
